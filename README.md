# 🍎 Controle de Laboratório de Alimentos - SENAI AP

Sistema desenvolvido para gestão de insumos, reserva virtual de estoque e controle de turmas.

### 🚀 Funcionalidades
- **Login:** Acesso restrito para Administradores e modo Leitura para Visitantes.
- **Inventário:** Cadastro de produtos com alerta de estoque mínimo e validade.
- **Reserva Virtual:** Bloqueio inteligente de ingredientes para aulas agendadas.
- **Logs:** Histórico de todas as ações realizadas no sistema.

### 🛠️ Tecnologias
- React.js (Vite)
- Tailwind CSS v4
- Firebase (Firestore & Auth)
- Lucide React (Ícones)
